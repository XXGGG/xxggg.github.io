## 🟢 偏移自身一半
```css
transform: translate(-50%, -50%);
```

## 🟢 去除a标签下划线
```css
text-decoration:none;
```
## 🟢去除列表的前缀符号
```css
list-style:none;
```
## 🟢字间距
```css
letter-spacing:20px;
```
## 🟢 不同的光标 
```css
cursor:pointer;
```
## 🟢 只留文字的背景：
```css
-webkit-background-clip:text;
```
## 🟢 对比度
```css
filter:contrast(30);
```
## 🟢 高斯模糊
```css
filtar:blur(20px);
```
## 🟢 延迟动画
```css
animation-delay:0.5s;
```
## 🟢 切割空间
```css
clip-path:inset(0px 50% 0 0);
```
## 🟢 隐藏背面
```css
backface-visibility:hidden;
```
## 🟢 设定透视深度
```css
perspective:(1000px);
```
## 🟢 img图片不被拖动
```css
-webkit-user-drag: none;
```
