# 🟢 什么是【Ajax】

其实只要知道ajax是（异步Aysnc的Js And Xml）AJAX就好了，

可以理解ajax就是在不重新加载页面的情况下，用js请求来的数据渲染一下新的页面，  
也就是比如点击一个按钮 异步请求来数据，然后用js对页面做一些改变
不用刷新页面。 

所谓的XMLHttpRequest 就是发送数据/请求数据用的  
XHR俗称 小黄人


对于vue 之前官方推荐的就是axios 也就是vue-axios...
